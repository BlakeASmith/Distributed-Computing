Diggind in to the gRPC example code¶

Installing gRPC¶

see the quickstart for more detail

First you need to install the gRPC module for go

Install Go Module for gRPC¶
$ export GO111MODULE=on # this forces go to use Go modules instead of GOPATH
$ go get google.golang.org/grpc@v1.28.1

The GO111MODULE situation is a bit of mess, this article explains it well https://dev.to/maelvls/why-is-go111module-everywhere-and-everything-about-go-modules-24k

Then install the protoc compiler If you’re on Arch Linux there is a package for it

Install protobuf from AUR¶
$ sudo pacman -Syu protobuf
